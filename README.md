# Web3 Portfolio Tracker

This project tracks a crypto portfolio.
It reads wallet balances and token values.
You can see total value in USD.

This project is made for beginners.
All code is simple and clear.

## Features
- Track ETH and ERC20 tokens
- Read wallet balances
- Show total value in USD
- Simple command line tool

## Requirements
- Node.js installed
- RPC provider account
- Free price API

## Setup
1. Clone this repo
2. Open folder in terminal
3. Run: npm install
4. Create a file called .env
5. Add RPC URL and API key
6. Run: node portfolio.js

## Example
Put wallet address in wallets.txt
Run the script.
Portfolio value will show.

## Learning Goals
- Learn portfolio logic
- Learn token balances
- Learn price conversion
